# Contributing to RAL

We welcome contributions from the community! Whether you are a developer, a translator (Kazakh/Russian), or a dairy specialist, your help is valuable.

## How to Contribute
1. **Report Bugs:** Open an issue on GitHub.
2. **Localization:** Help us translate the interface into Kazakh and other regional languages.
3. **Code:** Submit Pull Requests for offline-sync improvements or blockchain integration modules.

## Code of Conduct
We follow the Contributor Covenant. Please be respectful and inclusive.
